# Python Port Scanner

A simple TCP port scanner written in Python.

## Features

* Scan ports from 1 to 1024
* Detect open and closed ports
* Hostname to IP resolution
* Scan execution time measurement

## Usage

```bash
python portscanner.py
```

Then enter the target hostname or IP.

Example:

```
Enter target IP or hostname: localhost
```

## Technologies

* Python
* Socket programming
* TCP networking
